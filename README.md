# Право olymp — подготовка к олимпиадам по праву

Интерактивная статическая платформа (SPA) для подготовки к олимпиадам по праву, которые дают льготы при поступлении на программу **«Юриспруденция»** в НИУ ВШЭ и другие вузы.

Контент формируется на основе презентаций курсов **«Общее дело»** и официальных сайтов олимпиад.

## Что умеет

- **Олимпиады** — карточки с информацией о 15 перечневых олимпиадах (Высшая проба, ВСОШ, Ломоносов, СПбГУ, Кутафинская, ФЕМИДА и др.).
- **ВУЗы** — 17 вузов, разбитых на категории: топовые, сильные юридические, хорошие региональные. Для каждого указаны вступительные, релевантные олимпиады и примерные проходные баллы ЕГЭ на бюджет.
- **Темы** — 16 учебных тем по ТГП (Теория государства и права) из презентаций «Общее дело».
- **Тесты** — 16 тестов по одному к каждой теме с мгновенной проверкой и объяснениями.
- **Прогресс** — изученные темы и результаты тестов сохраняются в `localStorage`.
- **Адаптивный дизайн** — работает на телефоне и компьютере.

## Как добавить новые материалы

Все учебные данные находятся в `js/data.js`:

- `olympiads` — информация об олимпиадах.
- `universities` — информация о вузах с категориями и проходными баллами.
- `topics` — учебные темы. Каждая тема содержит `category`, `title`, `source`, `summary`, `content` (HTML), `tags`.
- `tests` — тесты. Каждый тест привязан к теме через `topicId` и содержит массив вопросов с вариантами ответов.

Подробная инструкция — в `PROJECT.md`.

## Как выложить на GitHub Pages

1. Создай новый репозиторий на GitHub.
2. Загрузи в него все файлы из этой папки.
3. Перейди в **Settings → Pages**.
4. В разделе **Build and deployment** выбери:
   - **Source:** Deploy from a branch
   - **Branch:** main, папка **/(root)**
5. Нажми **Save**. Через 1–2 минуты сайт будет доступен.

## Локальный запуск

```bash
python3 -m http.server 8000
```

Затем открой `http://localhost:8000`.

## Автоматическая проверка олимпиад

GitHub Actions workflow `.github/workflows/check_olympiads.yml` раз в неделю проверяет сайты олимпиад. Если сайт изменился или недоступен — создаёт issue в репозитории.

Чтобы включить: **Settings → Actions → General** → разрешить Actions, затем во вкладке **Actions** включить workflow «Проверка олимпиад».
