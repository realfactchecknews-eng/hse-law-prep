# Право olymp — подготовка к олимпиадам по праву

Интерактивная статическая платформа для подготовки к олимпиадам по праву, которые дают льготы при поступлении на программу **«Юриспруденция»** в НИУ ВШЭ.

## Что умеет

- **Олимпиады** — карточки с информацией о Высшей пробе, ВСОШ и других олимпиадах из Перечня РСОШ, принимаемых ВШЭ.
- **Темы** — учебные материалы из презентаций курсов (добавляются по мере поступления презентаций).
- **Тесты** — проверка знаний с мгновенной обратной связью и объяснениями.
- **Прогресс** — изученные темы и результаты тестов сохраняются в `localStorage`.
- **Поиск и фильтры** — удобный поиск по темам.
- **Адаптивный дизайн** — работает на телефоне и компьютере.

## Как добавить новые материалы

Все учебные данные находятся в `js/data.js`:

- `olympiads` — информация об олимпиадах.
- `topics` — учебные темы. Каждая тема содержит `category`, `title`, `source`, `summary`, `content` (HTML), `tags`.
- `tests` — тесты. Каждый тест привязан к теме через `topicId` и содержит массив вопросов с вариантами ответов.

## Как выложить на GitHub Pages

1. Создай новый репозиторий на GitHub.
2. Загрузи в него все файлы из этой папки (`index.html`, `css/`, `js/`, `README.md`, `.gitignore`).
3. Перейди в **Settings → Pages**.
4. В разделе **Build and deployment** выбери:
   - **Source:** Deploy from a branch
   - **Branch:** main, папка **/(root)**
5. Нажми **Save**. Через 1–2 минуты сайт будет доступен по адресу `https://<username>.github.io/<repo-name>/`.

## Локальный запуск

Просто открой `index.html` в браузере. Можно также использовать любой локальный сервер, например:

```bash
python3 -m http.server 8000
```

Затем открой `http://localhost:8000`.
